# Wilson Docs
Wilson documentation based on VitePress

## Installation
Install pnpm
```bash
$ npm install -g pnpm
```
Download dependencies
```bash
$ pnpm install
```

## Development

Development
```bash
$ pnpm docs:dev
```
Preview
```bash
$ pnpm docs:preview
```
Build
```bash
$ pnpm docs:build
```
